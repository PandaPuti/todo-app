# TODO List App

A simple browser-based TODO list application built with plain HTML, CSS, and JavaScript. It stores tasks locally in the browser using `localStorage`, so tasks persist across page refreshes.

## Features

- Add a new todo item by typing in the input and pressing Enter or clicking the ADD button
- Mark todos as completed using custom checkboxes
- Delete todos using the delete button
- Persist todos in the browser with `localStorage`

## Project Structure

- `index.html` - main page markup
- `style.css` - app styles and layout
- `app.js` - todo app logic, event handling, and storage
- `icons/` - project icons and related assets

## Getting Started

1. Open `index.html` in a web browser.
2. Type a todo item into the input field.
3. Press Enter or click the `ADD` button to add the item.
4. Use the checkbox to mark items complete.
5. Click the trash icon button to delete items.

## Notes

- No build tools or dependencies are required.
- Works entirely in the browser.
- Todo items are stored in browser `localStorage`, so they remain after refreshing the page.
